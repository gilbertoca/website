Help and Support

Stack Overflow

If you have a question on how to accomplish a goal with the Konik library or have a different how to do X question. Please see first if a similar question has been asked before. If this is not the case you create a new question. We are monitoring the Konik tags and will answer your questions.

Bugs and Features

Found a bug? The fastest way to get it fixed is to raise a ticket on GitHub. You can also see if a similar ticket already exist and comment on that. We try our best to provide a fix for the next release.

The ticketing system is also a perfect place to request a new feature.
