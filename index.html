Konik is an open source library implementing the ZUGFeRD invoicing data model.

Konik contains everything you need for an easy invoicing in and output.

Get started!

Open Source

First open source library to easily crate, read and validate ZUGFeRD invoices for the JVM. Licensed under AGPL 3.0.

Beyond the obvious

Easy understandable API which is also well documented. Validation of the invoice against the ZUGFeRD defenition and country specific regulations allows you process the invoice without suprises.

With options to choose

Easy way of invoice appending or extracting. Multiple PDF Carriages allows you to choose the perfect on for your needs.

Quick Start

               
   <dependencies>
       <dependency>
           <groupId>io.konik</groupId>
           <artifactId>konik</artifactId>
           <version>0.0.1</version>
       </dependency>
   </dependencies>

The easiest way to get started using konik in your project is with a dependency management system that is able to access the Maven Central Repository.

Copy the code snippet in the black box to your projects pom file.
